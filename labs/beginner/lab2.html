    Part (5): Creating Profile of Keychain Holder Head (10 minutes)

    1. Create an Axis Through the Cylinder:
      • Go to the Construct command in the top menu.
      • Select Axis Through Cylinder.
      • Click on the cylinder to establish the axis of rotation.
    2. Create a Plane at an Angle to the Axis:
      • Again, in the Construct menu, select Plane at Angle.
      • Click on the axis you created.
      • Enter the angle as "0" for the plane, then confirm.
    3. Use the Plane to Sketch the profile for the Top Holder on [üö® REMIX OPPORTUNITYüö®]:

      • To ensure we remain within bounds, go to Create, then Project, Select Entity (Selection Filter) and click on the perimeter of the top circle of the cylinder for Geometry. Kepp Projection Link on.
      • Your Design Constraints for your REMIX are as follows:
      • Draw the profile you would be revolving for the top holder on this plane. Recall that this should be symmetric and should not extrude out too much. Make sure that the profile is closed!

    Part (6): Revolving Radially Symmetric Geometry(10 minutes)

    Video Summary:
    1. Revolve Command: Go to Create > Revolve in the toolbar.
    2. Select Profile: Click on the profile (the closed shape) that you want to revolve.
    3. Select Axis: Click the axis of symmetry (the cyliner axis we constructed) around which the profile will revolve.
    4. Set Angle: In the Revolve dialog, set the angle to 360¬∞ (or the desired angle for partial revolution if you want to play around with this!).
    5. OK: Click OK to complete the operation.

    Part (7): Making Hole for Keychain (5 minutes)

    1. Create a Tangent Plane: Create a plane that is tangent to the surface of your revolved part using Construct > Tangent Plane.
    2. Sketch a Circle on the Tangent Plane: Start a sketch on the tangent plane and draw a circle where you want the hole.
    3. Dimension the Circle: Use the Dimension tool to set the circle's diameter to 4mm (or your desired size).
    4. Extrude Cut the Hole: Use Create > Extrude, select the circle, and set it to Cut with the Symmetric direction to cut both sides of the part.
    5. Finish: Click OK to complete the hole creation process.
