3D printing is an additive (add material) manufacturing process where computer-aided G-code (instruction code for the machine) is translated to build up material layer by layer. It has various techniques, including FDM , which extrudes hot thermoplastics (plastics that melt at elevated temperatures), SLA, which uses photoresin polymers that cure via light, and Polyjet 3D printing, among others.

    Part (5): Creating Profile of Keychain Holder Head (üö®REMIXüö®) (20 minutes)

    1. Create an Axis Through the Cylinder:

      • Draw the profile you would be revolving for the top holder on this plane. Recall that this should be symmetric and should not extrude out too much. Make sure that the profile is closed!

    Part (6): Revolving Radially Symmetric Geometry(5 minutes)

    Video Summary:
    1. Revolve Command: Go to Create > Revolve in the toolbar.
    2. Select Profile: Click on the profile (the closed shape) that you want to revolve.
    3. Select Axis: Click the axis of symmetry (the cyliner axis we constructed) around which the profile will revolve.
    4. Set Angle: In the Revolve dialog, set the angle to 360¬∞ (or the desired angle for partial revolution if you want to play around with this!).
    5. OK: Click OK to complete the operation.

    Part (7): Making Hole for Keychain (5 minutes)

    1. Create a Tangent Plane: Create a plane that is tangent to the surface of your revolved part using Construct > Tangent Plane.
    2. Sketch a Circle on the Tangent Plane: Start a sketch on the tangent plane and draw a circle where you want the hole.
    3. Dimension the Circle: Use the Dimension tool to set the circle's diameter to 4mm (or your desired size).
    4. Extrude Cut the Hole: Use Create > Extrude, select the circle, and set it to Cut with the Symmetric direction to cut both sides of the part.
    5. Finish: Click OK to complete the hole creation process.

    Part (13): Export STL Files & Share Fusion Project (10 minutes)

    We would like to export each part individually. To do so:
    1. Go to the hierarchy in Fusion 360, right-click on the component or body, and click Export.
    2. Select the file type as STL and save it locally. Use the naming convention: "LastName_FirstName_Number" (e.g., Jane_Doe_1).
    3. Repeat the same steps for all components you have.
    4. Upload all STL files to your Google Drive.
    5. Share the link to your Fusion 360 file so your instructors can access and edit your files if necessary. To do this:
      1. Open your project in Fusion 360.
      2. Click on the File menu and select Share > Share File Link.
      3. Enable the option to allow downloading
      4. Copy the generated link and share it with your instructors.

    OPTIONAL--Part (14): Directly Export for 3D Printing

    1. Export the 3D Model:
      • Click File in the top-left corner, then select 3D Print.
    2. Use the 3D Print Utility:
      • Click on 3D Print from Utility.
      • Choose Application Custom, and then load CURA from your applications list.
    3. Open in Cura:
      • After selecting your model, click OK.
      • CURA will open automatically with the selected model.
    4. Export Each Component:
      • Repeat the process for each component to export them one by one.

    OPTIONAL--Part (15): Setting Up File for 3D Printing

    1. Adjust Model Orientation:
      • Ensure your models are in the correct orientation.
      • Use Rotate and Move tools to adjust the position if needed.
      • Make sure models do not overlap.
    2. Adjust 3D Printing Settings:
      • Set Layer Thickness to 0.1mm (fine quality, for threading).
      • Set Speed to 100mm/s.
      • Set Fill Density to 15%.
      • Enable Support (set to "only touching the print bed").
      • Disable Build Plate Adhesion (usually recommend to use brim, but here we just don't want to affect your design).
    3. Slicing and Preview:
      • Click Slice to generate the G-code.
      • Click Preview to check your print setup before exporting.
